== Description ==

The PHP Live! WordPress plugin provides a simple method to add live chat to your WordPress pages. The plugin does not replace the actual PHP Live! system. The plugin is just a method to paste the PHP Live! HTML Code to your WordPress pages.

[ Important ]
You must already have an existing PHP Live! system to utilize the WordPress plugin.  The WordPress plugin is just a method to paste the PHP Live! HTML Code to your WordPress pages to display the chat icon. If you are new to PHP Live!, please visit our website for more information:

https://www.phplivesupport.com

**About PHP Live!**

PHP Live! is a PHP and MySQL powered live chat software for websites.

== Installation ==

1. Upload the 'php-live-wordpress' directory to your Wordpress plugins directory ('/wp-content/plugins/')
2. Activate the plugin in the "Plugins" admin panel using the "Activate" link.
3. Click the newly created 'PHP Live!' menu on the left side of the WordPress admin panel.
4. Proceed with the steps detailed on the PHP Live! plugins page.
